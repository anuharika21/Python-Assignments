# Python Assignments 📝

This repository contains Python assignment solutions covering Strings, Lists, and Dictionaries —
practiced with conditional statements and control flow for a strong programming foundation.

---

## 📁 Assignment Structure

### 🔤 1. Python Strings
- String creation, indexing & slicing
- String methods (upper, lower, strip, replace, split, join)
- String formatting & f-strings
- Conditionals on strings (palindrome check, vowel counter)
- Looping through characters

### 📋 2. Python Lists
- List creation, indexing & slicing
- List methods (append, remove, pop, sort, reverse)
- Iterating with for & while loops
- Conditional filtering (even/odd, positive/negative)
- Nested lists & list comprehension

### 📚 3. Python Dictionary
- Dictionary creation & key-value access
- Dictionary methods (keys, values, items, get, update)
- Looping through dictionaries
- Conditional logic on dictionary values
- Nested dictionaries
